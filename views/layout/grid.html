Grid
  • Column Width
  • Margins

Factory tools are implement a 12-column grid system. The reasons are:

  • It creates consistency between layouts.
  • It is flexible. It makes easy to evenly distribute a column, regardless of whether you want to create a 2 (spanning of 6), 3 (spanning of 4), or 4 (spanning of 3) content column.
  • It is mobile ready.

Column Width

Column width is calculated in percentage. Since it is a 12-column grid system, it has a total of 12 spanning columns, and the width of each spanning column is calculated by using the number of its spanning and divide by 12.

Margins

The responsive grid also focuses on consistent margins. Margins follow a multiples of 8 in px, so they can be 8, 16, 24, etc. However, sometimes you may want to implement a flexible margin, instead of a fixed amount, on a column. In order to do that, we implement a responsive margin system, and give it a name called offset.
