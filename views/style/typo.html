Style

Too many font sizes in one page will mess up the layout, and also destroy the viewers visual experience. Therefore, it is a nice thing to set some standards for the font size, and assign them with a meaningful name so that we know when to use a certain font size.


The basic set of font size are based on a scale of 12, 14, 20, and 26. And the set of line height are based on a scale of 22, 24, 30, and 36.

title bold 26px(font size) 36px(line height)

sub title regular 20px(font size) 30(line height)

body regular 14px(font size) 24px(line height)

caption light italic 12px(font size) 22(line height)

button regular all cap 14px(font size) 24(line height)
